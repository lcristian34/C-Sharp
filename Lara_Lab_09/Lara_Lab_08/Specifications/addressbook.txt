The Main form will have a menu strip that can acces a help button with info.
It will halso have a form that allows the user to search by first name and last name.
At the end of the program the info of the list is saved to the file again. Updating the info.
The application will have printing capavilities as well as a print prevew. The print preview 
has a determinated number of records that it can hold per page.
